from LAC import LAC

def lac_username(sentences: str) -> list:
    # 装载LAC模型
    user_name_list = []
    lac = LAC(mode="lac")
    lac_result = lac.run(sentences)
    for index, lac_label in enumerate(lac_result[1]):
        if lac_label == "PER":
            user_name_list.append(lac_result[0][index])
    return user_name_list

## 提取编号

设计思路：

1. 使用正则表达式匹配的方式去提取

```
[A-Za-z]*\d+(?!\年|\月|\日|时\分\秒)
```

这个正则表达式的含义如下：

- `[A-Za-z]*`：匹配零个或多个大小写字母（编号名的字母部分）。
- `\d+`：匹配一个或多个数字（编号名的数字部分）。
- `(?!\年|\月|\日)`：否定断言（negative lookahead），表示匹配的数字后面不能跟着"年"、"月"或"日"。



正则表达式中的元字符和符号的含义如下：

- `[A-Za-z]`：表示匹配任意一个大小写字母。
- `*`：表示匹配前面的元素零次或多次。
- `\d`：表示匹配任意一个数字。
- `+`：表示匹配前面的元素一次或多次。
- `(?!\年|\月|\日)`：否定断言，用来排除指定的后续文本。



因此，这个正则表达式可以用来匹配满足以下条件的文本：

- `[A-Za-z]*`：匹配零个或多个大小写字母（编号名的字母部分）。
- `\d+`：匹配一个或多个数字（编号名的数字部分）。
- `(?!\年|\月|\日)`：排除以"年"、"月"或"日"结尾的情况。换句话说，匹配的数字不能紧跟着"年"、"月"或"日"。



2. 同时，因为编号只在最后出现，在提取到的数据中只拿最后一个编号即可

```python
if matches:
    last_match = matches[-1]
    print(last_match)
else:
    print("未找到匹配的编号")
```





完整的demo实例

```python
import re

text = "六支队（原二支队）王小青报销2023年5月3日黄晴，徐志胜经营指导FG135餐费"
pattern = r"[A-Za-z]*\d+(?!\年|\月|\日|时\分\秒)" # 使用了负向先行断言，排除掉和时间有关的数字
matches = re.findall(pattern, text)


if matches:
    # 取最后一个编号信息
    last_match = matches[-1]
    print(last_match)
else:
    print("未找到匹配的编号")
```

def read_excel(file_path):
    """
    读取Excel文件的函数
    :param file_path: 文件路径
    :return: 二维列表，包含读取到的所有单元格的值
    """
    excel = win32.gencache.EnsureDispatch('Excel.Application')
    workbook = excel.Workbooks.Open(file_path)
    worksheet = workbook.ActiveSheet

    # 获取行数和列数
    rows = worksheet.UsedRange.Rows.Count
    columns = worksheet.UsedRange.Columns.Count

    data = []
    for row in range(2, rows + 1):  # 从第二行开始读取
        row_data = []
        for col in range(1, columns + 1):
            cell_value = str(worksheet.Cells(row, col).Value)
            row_data.append(cell_value)
        data.append(row_data)

    workbook.Close(False)
    excel.Quit()

    return data


def write_excel(file_path, data):
    """
    将数据写入Excel文件的函数
    :param file_path: 文件路径
    :param data: 二维列表，包含要写入的数据
    """
    excel = win32.gencache.EnsureDispatch('Excel.Application')
    workbook = excel.Workbooks.Add()
    worksheet = workbook.ActiveSheet

    # 写入表头
    header = ['序号', '摘要', '金额', '编号', '姓名']
    for col_idx, cell_value in enumerate(header):
        worksheet.Cells(1, col_idx + 1).Value = cell_value

    for row_idx, row_data in enumerate(data):
        for col_idx, cell_value in enumerate(row_data):
            worksheet.Cells(row_idx + 2, col_idx + 1).Value = cell_value

    workbook.SaveAs(file_path)
    workbook.Close(True)
    excel.Quit()

## **问题**

**问题：读写效率低，读数据：6600条 - 30min** 

**原因：**

1. **用`pywin32`库进行Excel文件的读写操作可能会比较慢，因为它是通过调用Microsoft Excel的COM接口来实现的。COM接口的调用涉及到进程间通信，因此会有一定的开销。**
2. **名字识别函数执行时间较长，导致整体的效率不高。**



**解决方案：**

1. **使用pandas代替pywin32**
2. **将`lac = LAC(mode="lac")`移出`identity_name`函数。在函数外部定义一个全局的`lac`对象，并在程序启动时进行初始化。这样可以避免每次调用`identity_name`函数时都重新加载模型，提高效率。**

## **其他**

问：python操作Excel，需要支持后缀名为xlsx、xls、et格式

答：要在Python中操作Excel文件，你可以使用第三方库，如`openpyxl`和`xlrd`，来处理后缀名为`.xlsx`和`.xls`的Excel文件。这些库提供了许多功能，如读取和写入Excel文件中的数据、创建和修改工作表、处理单元格格式等。

对于后缀名为`.et`的Excel文件，这是一种由WPS表格创建的特殊格式，它不是标准的Excel文件格式。目前，没有公开的Python库可以直接读取和写入`.et`文件。如果你需要处理`.et`文件，一种解决方案是将其转换为其他常见的Excel格式，如`.xlsx`或`.xls`，然后使用相应的Python库进行操作。
